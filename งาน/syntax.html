ไวยากรณ์และตัวแปรในภาษา C

กลับไปหน้าหลัก

ภาษา C มีไวยากรณ์ที่ค่อนข้างเข้มงวด ซึ่งรวมถึงการใช้เครื่องหมายจุดทศนิยม (;) เพื่อสิ้นสุดคำสั่ง และการใช้วงเล็บปีกกา ({}) เพื่อกำหนดบล็อกของโค้ด...

ตัวแปรในภาษา C

ตัวแปรในภาษา C ต้องถูกประกาศก่อนใช้งาน โดยระบุชนิดข้อมูล เช่น int, float, char เป็นต้น...

 
#include <stdio.h>
int main() {
    int age = 25; // ประกาศตัวแปรชนิด int
    float height = 175.5; // ประกาศตัวแปรชนิด float
    char grade = 'A'; // ประกาศตัวแปรชนิด char

    printf("Age: %d\n", age);
    printf("Height: %.2f\n", height);
    printf("Grade: %c\n", grade);

    return 0;
}
